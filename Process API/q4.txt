/*
 * 总结来说，exec 系列函数的主要区别在于：
 * 参数的传递方式（单个参数列表或参数数组）。
 * 是否在 PATH 中搜索程序（p 在函数名中表示这个行为）。
 * 是否允许指定环境变量（e 在函数名中表示这个行为）。
 * 程序路径的指定方式（直接路径或名称）

 * 每个 exec 函数变体的主要区别在于它们如何处理程序的路径、参数列表和环境变量。
 * 开发者可以根据具体需求选择最合适的函数变体。
 * 例如，如果需要设置特定的环境变量，可以使用 execle 或 execve；
 * 如果需要在 PATH 中搜索程序，可以使用 execvp 或 execlp。
